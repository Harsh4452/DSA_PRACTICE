# 📘 DSA Practice

Minimal repo to track my DSA / LeetCode solutions with short notes.

# If starting a brand new repo
git init

# If cloning an existing repo from GitHub
git clone https://github.com/<username>/<repo>.git
cd <repo>

# Check status of repo
git status

# Stage changes (add all modified/new files)
git add .

# Commit with message
git commit -m "Added <problem name> solution"

# Push changes to GitHub
git push origin main
